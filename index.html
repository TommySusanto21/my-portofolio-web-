What I've Built

Projects

OceanLens – Aplikasi Agregator Berita Mobile

Mengembangkan OceanLens, aplikasi mobile agregator berita yang mengintegrasikan berita dari berbagai media online dalam satu platform. Aplikasi ini dilengkapi fitur pengelompokan topik, analisis sentimen berbasis machine learning, serta ringkasan berita untuk membantu pengguna memperoleh informasi secara lebih cepat dan mudah dipahami.

OILSARAI – Segmentasi Tumpahan Minyak Menggunakan Deep Learning

Mengembangkan OILSARAI, aplikasi berbasis web untuk mendeteksi dan melakukan segmentasi tumpahan minyak pada citra Synthetic Aperture Radar (SAR) menggunakan metode deep learning. Sistem ini memungkinkan pengguna mengunggah citra SAR, menjalankan proses segmentasi, serta membandingkan hasil dari beberapa model seperti U-Net, U-Net++, dan DeepLabV3+.

Website Destinasi Wisata Maritim

Mengembangkan website destinasi wisata maritim sebagai proyek mata kuliah Interaksi Manusia dan Komputer (IMK). Website ini dirancang untuk memperkenalkan berbagai destinasi wisata bahari di Provinsi Kepulauan Riau dengan antarmuka yang menarik, responsif, dan mudah digunakan oleh pengguna.

SILADAS – Sistem Layanan Daftar Arsip Statis

Mengembangkan SILADAS, sistem informasi berbasis web untuk layanan daftar arsip statis selama menjalani program magang di Dinas Perpustakaan dan Kearsipan Kota Tanjungpinang. Sistem ini memudahkan pengguna dalam mencari, mengakses, dan menelusuri informasi arsip statis secara cepat dan efisien..

AR Wayfinding

AR Wayfinding merupakan aplikasi berbasis Augmented Reality (AR) yang dikembangkan untuk membantu pengguna menemukan lokasi penting di lingkungan SMAN 5 Tanjungpinang secara lebih mudah dan interaktif. Aplikasi ini memanfaatkan teknologi AR untuk memberikan petunjuk arah menuju fasilitas seperti toilet, ruang guru, dan kantin melalui perangkat smartphone.

Mesin Turing

Simulasi Mesin Turing matematika sederhana untuk operasi penjumlahan dan pengurangan yang berhasil dijalankan dengan baik.
